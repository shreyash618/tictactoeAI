------------------------------------------------------------------------
We created a graphical Tic Tac Toe game where a human plays against
a artifical intelligence. The game offers three modes, easy, medium and 
hard. In easy mode, the computer plays a random move against the human
player. In medium mode, the computer tries to win while simultaneously
block the human player's wins. In hard mode, the computer uses the minimax
algorithm. We created a graphical user interface (GUI) using Java Swing and
AWT packages to play the game.
------------------------------------------------------------------------
PROJECT TITLE: Tic Tac Toe

PURPOSE OF PROJECT: A AI plays a game of tic tac toe against a human on a GUI.

VERSION or DATE: 05/21/2022

HOW TO START THIS PROJECT: Compile the TicTacToeGrid.java program and 
click the "Start" button to play the game.

AUTHORS: Shreya Shukla, Staney Yu

USER INSTRUCTIONS:
1) Compile the Java program in the JVM.
2) Select a piece: "Piece X" or "Piece O".
3) Select what player you want to be: "Player 1" (if you want to go first)
or "Player 2" (if you want to go second).
4) Select a mode: "easy", "medium", and "hard".
5) Click "Start" to begin the game.
6) Click the buttons on the tic tac toe grid corresponding to where you want to 
place a piece.
7) When the game is over, a pop up window will show up showing who is the winner.
8) To replay the game, click "Clear" to clear the board and then repeat the instructions.
